# Архитектура

## Границы модулей

- `providers/` реализует внешние границы. Локальные реализации работают без сети; HTTP-реализации включаются явно.
- `domain/` содержит типы и runtime-схему недоверенного LLM-ответа.
- `services/` выполняет однотипные этапы: recovery, chunking, TTS, WAV aggregation, subtitles, render и manifest.
- `pipeline/` связывает этапы и отвечает за каталог конкретного запуска.
- `remotion/` получает только сериализуемые `RenderProps` и не знает о providers.

## Обработка ошибок

Невалидный сценарий не доходит до TTS. Orchestrator делает не более трёх попыток: сначала `generate`, затем `repair` с сообщениями Zod. Сетевая ошибка provider также попадает в bounded recovery. После исчерпания попыток pipeline завершается с явной ошибкой.

После каждого завершённого этапа конвейер атомарно обновляет манифест. Повтор с тем же `RUN_ID` проверяет размер и SHA-256 артефактов: валидные результаты переиспользуются, а повреждённый этап и его зависимости выполняются заново. Поле `attempts` увеличивается только при фактическом запуске этапа, поэтому восстановление можно отличить от повторного вычисления.

Граф зависимостей описан отдельно от последовательности выполнения. Решение для этапа фиксируется как `reused` или `executed` вместе с причиной: проверенные артефакты, отсутствующий файл, нарушение целостности, незавершённый статус или изменение вышестоящего этапа. Это позволяет диагностировать восстановление без анализа журналов процесса.

## Timing

Каждый TTS chunk возвращает `durationMs`. Heuristic fallback размещает cues последовательно и без перекрытий; последний `endMs` определяет длительность Remotion-композиции. Внешний TTS обязан вернуть длительность в `x-audio-duration-ms`, поэтому неявной оценки для production-подобного provider нет.

## Доверительные границы

- LLM JSON — недоверенный ввод, проверяется Zod.
- TTS WAV — недоверенный бинарный ввод, проверяется RIFF header перед склейкой.
- Секреты существуют только в окружении процесса и не записываются в manifest.
- `artifact-manifest.json` содержит только относительные пути, размеры и SHA-256.
