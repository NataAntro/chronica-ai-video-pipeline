`npm audit` после обновления зафиксированных версий сообщает `0 vulnerabilities`.

## Ассеты

- `docs/architecture.svg` создана с нуля для этого репозитория и не содержит внешних изображений.
- `docs/assets/telegram-product.png` — композиция из публичной карточки Telegram-бота и интерфейса Chronica.
- `docs/assets/publisher-dashboard.png` — снимок панели Chronica Public Publisher без секретов и персональных данных.
- `docs/assets/generated-video.gif` — фрагмент видео, созданного медиаконвейером Chronica.

Продуктовые материалы предоставлены автором Chronica. Для README подготовлены отдельные производные файлы; исходные медиа в репозиторий не входят.

- В Remotion-композиции используются только CSS-формы, градиенты и системный Arial.
- Локальная озвучка генерируется математически во время запуска и не является записью человека или музыкой.
- Текст технологического дайджеста создан специально для этого репозитория.
- WAV/MP4, создаваемые локальным демонстрационным запуском, в публикационный набор не входят.
